#### np.newaxis

源码中，np.newaxis是None的别名。

```python
a = np.arange(3) 			# array([0,1,2])
b = a[:, np.newaixs] 	# array([[0], [1], [2]])
c = a[:, None] 				# array([[0], [1], [2]])
```
